


# **# Installation**  
     1. clone the repo
     2. npm run install
     3. npm run dev

 - ***Frontend*** - React & Vite & typescript (mui for design system, redux-toolkit mange state app).
 - ***Backend*** - Node & express & typescript.
 - ***DB*** - mongoDB (Mongoose for ODM).
